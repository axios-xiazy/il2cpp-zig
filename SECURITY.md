# Security Policy

| Version | Supported          |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please follow these steps:

1. **Do not open a public issue** - this could put the security of the project at risk
2. Send an email to [INSERT EMAIL ADDRESS] with the details of the vulnerability
3. Include the following information in your report:
   - Description of the vulnerability
   - Steps to reproduce the issue
   - Potential impact of the vulnerability
   - Any suggested fixes (if applicable)

## Response Timeline

- You will receive an acknowledgment of your report within 48 hours
- We will investigate the issue and provide an initial response within 1 week
- Updates on the progress will be provided every 2 weeks until resolution
- Once confirmed, we will work to address the vulnerability as quickly as possible

## Security Best Practices

When using this tool, please keep in mind:

- Only use it on files you own or have explicit permission to analyze
- Respect intellectual property rights and applicable laws
- Be mindful of privacy concerns when sharing analysis results
- Keep your local copies secure and up to date
